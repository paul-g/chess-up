virtual MovesList Piece::possibleMoves ( )
pure virtual

Returns a list of possible moves for this piece. A move is 'possible' if and only if it:

  1. corresponds to the movement pattern of the piece
  2. is within the bounds of the board
  3. it is not blocked by the placement of other pieces Additional validity checks should be performed in validMoves().
